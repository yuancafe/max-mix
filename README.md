# max-mix

EN: A practical MiniMax multimodal skill pack for IDE/CLI agents.  
中文：一个面向 IDE/CLI Agent 的 MiniMax 多模态技能包。

---

## 1) What’s Included | 包含内容

- `max-image-gen` - image generation (`image-01`) | 图片生成
- `max-video-gen` - video generation (Hailuo) | 视频生成
- `max-tts` - text-to-speech (`speech-hd`) | 文本转语音
- `max-voice-clone` - upload + clone voice | 音色上传与复刻
- `max-music-gen` - music generation (`music-2.5/2.6`) | 音乐生成
- `max-music-cover` - cover generation (`music-cover`) | 翻唱生成
- `max-search` - web search (`coding-plan-search`) | 搜索
- `max-vision` - vision/OCR (`coding-plan-vlm`) | 图像理解/OCR
- `max-text-chat` - text/coding chat (`MiniMax-M*`) | 文本/编程对话

---

## 2) Install | 安装

```bash
bash ./install.sh
bash ./check-deps.sh
```

---

## 3) Authentication | 认证配置

EN: Authenticate once with mmx CLI.  
中文：先用 mmx CLI 登录一次。

```bash
mmx auth login --api-key <YOUR_MINIMAX_API_KEY>
mmx quota show
```

EN: Credential lookup order used by scripts:  
中文：脚本读取凭据顺序：

1. `MINIMAX_API_KEY` environment variable | 环境变量
2. `~/.mmx/config.json` generated by `mmx auth login` | `mmx auth login` 生成的本地配置

---

## 4) Usage Pattern | 使用模式

EN: Most generation skills support `plan -> run`.  
中文：大多数生成类技能支持 `plan -> run` 两阶段。

```bash
python3 skills/max-video-gen/scripts/max_video_gen.py plan --interactive
python3 skills/max-video-gen/scripts/max_video_gen.py run --plan-file <plan.json>
```

EN: `max-search` and `max-vision` are direct-first (can still use plan/run).  
中文：`max-search` 和 `max-vision` 默认可直接调用（也支持 plan/run）。

```bash
python3 skills/max-search/scripts/max_search.py --q "MiniMax latest updates"
python3 skills/max-vision/scripts/max_vision.py --image ./shot.png --prompt "Extract text"
```

## Model Coverage | 模型覆盖

EN: This skill pack is built on MiniMax Token Plan model families via `mmx` and MiniMax APIs.  
中文：本技能包基于 MiniMax Token Plan 模型族（通过 `mmx` 与官方 API 调用）。

| Skill | MiniMax model / capability |
|---|---|
| `max-text-chat` | `MiniMax-M*` (e.g. `MiniMax-M2.7-highspeed`) |
| `max-tts` | `speech-hd` (e.g. `speech-2.8-hd`) |
| `max-video-gen` | `MiniMax-Hailuo-2.3-6s-768p`, `MiniMax-Hailuo-2.3-Fast-6s-768p` |
| `max-music-gen` | `music-2.5`, `music-2.6`, `lyrics_generation` |
| `max-music-cover` | `music-cover` |
| `max-image-gen` | `image-01` |
| `max-vision` | `coding-plan-vlm` |
| `max-search` | `coding-plan-search` |
| `max-voice-clone` | MiniMax voice cloning APIs (runtime entitlement required) |
